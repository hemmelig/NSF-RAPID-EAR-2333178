# Collaborative Research: RAPID: A novel magnetometer network to capture the ongoing inflationary episode at Askja volcano, Iceland

This repository brings together the code, data, and metadata used to prepare the proposal for the funded [NSF RAPID project EAR-2333178](https://www.nsf.gov/awardsearch/showAward?AWD_ID=2333178), archive the data recovered from the field, produce the NSF reports, and some basic analysis. This award is linked with [EAR-2333180](https://www.nsf.gov/awardsearch/showAward?AWD_ID=2333180).
